<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Project</p>

```
git clone https://github.com/Kunal1001/chatter-box.git
```

<p>2. Install dependencies (do this for both: server and client)</p>

```
npm i
```

<p>3. Run server</p>

```
nodemon aap.js
```

<p>4. Run client</p>

```
npm run dev
```

<p>5. Enjoy 🌞</p>
